# Killer Queen

## Short Description
Jet upward through an infested planetary wound, kill the apex nests, and start the purge.

## One-Sentence Pitch
`Killer Queen` is a fast 2D jetpack action game about climbing through a living hive-city shaft, exterminating infestation anchors, and carving your way into the rest of a dying planet.

## Page Header Copy
Sanctuary still holds above the breach. The planet does not.

## Main Description
You are not escaping Khepri-9. You are opening it up.

This alpha is built around a sharp vertical slice:
- a live sanctuary front end
- a locked starter weapon choice
- Hive Shaft, the first vertical endurance purge
- Brood Warden, the first infestation apex
- Sunset Ruin Skyline, the second region reveal
- Spire Matriarch, the surface apex that proves the world is bigger than the shaft

The core loop is simple:
climb, kill anchors, expose the apex, cleanse the region, and push deeper into the planet.

## Feature Bullets
- Jetpack-driven vertical endurance platforming
- Melee-first combat with a starter weapon lock for the alpha
- Two complete alpha bosses with distinct region framing
- SNES/NES-informed command-console presentation
- Pixel-driven combat art pipeline with authored slash FX

## Controls
- `A / D` or `Left / Right`: move
- `Space`: jump
- `Left Shift`: jetpack
- `F` or `Left Mouse`: attack
- `R`: restart run
