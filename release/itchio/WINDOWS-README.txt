KILLER QUEEN
Planet Purge Alpha

CONTROLS
- A / D or Left / Right: move
- Space: jump
- Left Shift: jetpack burst
- F or Left Mouse: attack
- R: restart run

ALPHA CONTENT
- Sanctuary front end and starter weapon lock
- Region 1: Hive Shaft
- Boss 1: Brood Warden
- Region 2: Sunset Ruin Skyline
- Boss 2: Spire Matriarch

KNOWN NOTES
- This is an alpha build focused on vertical endurance, combat feel, and world direction.
- Controller prompts are supported.
- If a custom input profile feels wrong, use the in-game Systems menu to reset bindings.
